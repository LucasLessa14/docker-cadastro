# Aplicação de cadastro com Docker

**Version 2.0.0** - [Change log](CHANGELOG.md)

Aplicação simples de cadastros usando docker e docker-compose, exercício do [curso de docker](https://www.udemy.com/course/curso-docker/)

## Pré-requisitos
- docker
- docker-compose

## Tutorial

Subir os containers
```
docker-compose up -d
```
